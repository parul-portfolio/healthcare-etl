## Healthcare ETL & Data Quality Project

### Objective
Design a reliable ETL pipeline for healthcare admission data with a focus on:
- Data quality
- Standardization
- Analytics readiness

### Tech Stack
- Python (Pandas, NumPy)
- SQL-style transformations
- Power BI (dashboard layer) --in progress

### Key Features
- Duplicate detection
- Date validation
- Numeric & categorical standardization
- Data quality monitoring checks

### Outcomes
Clean dataset ready for:
- BI dashboards (proposed)
- KPI monitoring
- Stakeholder reporting
